Integrate with Windows

This section of the documentation provides developer guidance for integrating with Windows. It includes information on how to integrate with Windows system components, as well as other Windows integration features.

Windows system components that support 3rd party integration

The following table lists the Windows system components that support integration for 3rd party developers.

Feature Description
Feed providers Learn how to integrate into the Windows feeds experience.
Microsoft Copilot hardware key providers Learn how to register as an app that can be selected as the launch app for the Microsoft Copilot hardware key.
Search providers Learn how to integrate into the Windows Search experience.
Widget providers Learn how to implement a Windows widget service provider to support your app.

Other Windows integration features

The following table lists other Windows integration features that support 3rd party developers.

Feature Description
Cross Device Resume (XDR) using Continuity SDK Guidelines for first & third party developers to integrate with Windows XDR experiences using the Continuity SDK.
Smart App Control Smart App Control is a new app execution control feature that combines Microsoft’s app intelligence services and Windows' code integrity features to protect users from untrusted or potentially dangerous code.
Windows Share integration The Windows Share Sheet is a system-provided UI that enables users to share content from your app with other apps.
Windows People experiences Learn how third party apps can integrate with Windows People experiences using APIs to store their contacts.

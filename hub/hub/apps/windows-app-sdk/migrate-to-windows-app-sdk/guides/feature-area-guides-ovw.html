Overview of feature area guides

A collection of migration guidance topics, each focusing on a specific feature area.

Feature area guides in this section

Topic Description
User interface migration (including WinUI 3) This topic shows how to migrate your user interface (UI) code, including migrating to WinUI 3.
Windowing functionality migration This topic contains guidance related to window management, including migrating from UWP's ApplicationView/CoreWindow or AppWindow to the Window App SDK Microsoft.UI.Windowing.AppWindow.
Application lifecycle functionality migration This topic contains migration guidance in the application lifecycle feature area.
Toast notifications functionality migration This topic contains migration guidance in the toast notifications feature area.
Push notifications functionality migration This topic contains migration guidance in the push notifications feature area.
DirectWrite to DWriteCore migration DWriteCore is the Windows App SDK implementation of DirectWrite.
MRT to MRT Core migration This topic contains guidance for migrating from UWP's Resource Management System (also known as MRT) to the Windows App SDK's MRT Core.
Threading functionality migration This topic shows how to migrate your threading code.
Background task migration strategy This topic contains considerations and strategies for approaching the migration process, and how to migrate your UWP background tasks to use the Windows App SDK background task APIs.

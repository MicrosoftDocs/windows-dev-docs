Troubleshoot App creation

If you encounter a permission error when you attempt to create a new app in Partner Center, please ensure the following settings are configured correctly in your Microsoft Entra ID portal.

Enable app registrations

  1. Navigate to the User settings blade in Microsoft Entra ID.
  2. Ensure that the Users can register applications setting is set to Yes.

Assign Application developer permissions

  1. Navigate to the Roles and administrators blade in Microsoft Entra ID portal.
  2. Ensure that the user attempting to create a new app has Application developer permissions.

Create a new Microsoft Entra ID tenant

If the above steps do not resolve the error, create a new Entra ID tenant and associate it with your Partner Center account. For more information, see Associate Azure Entra ID with your Partner Center account.

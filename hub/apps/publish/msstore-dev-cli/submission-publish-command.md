# Submission - Publish Command

Starts the submission process for the existing Draft.

## Usage

```console
msstore submission publish <productId>
```

## Arguments

| Argument    | Description |
|-------------|-------------|
| `productId` | The product ID. |

## Help

```console
msstore submission publish --help
```

# Submission - Poll Command

Polls until the existing submission is PUBLISHED or FAILED.

## Usage

```console
msstore submission poll <productId>
```

## Arguments

| Argument    | Description |
|-------------|-------------|
| `productId` | The product ID. |

## Help

```console
msstore submission poll --help
```

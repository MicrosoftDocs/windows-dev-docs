# Submission - Status Command

Retrieves the current status of the store submission.

## Usage

```console
msstore submission status <productId>
```

## Arguments

| Argument    | Description |
|-------------|-------------|
| `productId` | The product ID. |

## Help

```console
msstore submission status --help
```

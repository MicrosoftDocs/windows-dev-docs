# Publish your app in the Microsoft Store

:::image type="content" source="images/header-store.png" alt-text="A briefcase icon with a grid pattern representing an app package, displayed against a soft blue gradient background. This header image symbolizes the Microsoft Store publishing process for Windows applications." border="false":::

---

Distributing your app through the Microsoft Store is a strong choice for apps of any type and size. The Store acts as a centralized destination where Windows users can discover and install apps, giving developers access to a global audience across Windows 10 and Windows 11.

The Microsoft Store provides flexible monetization options, allowing you to use your own commerce platform and choose a revenue model that fits your business. It also supports a broad range of app types and technologies, including traditional desktop apps, which can be published to the Store without requiring changes to your existing code or installer.

For a detailed overview of what the Microsoft Store offers, see [Benefits of distributing your apps via the Microsoft Store](https://go.microsoft.com/fwlink/?linkid=2278501).

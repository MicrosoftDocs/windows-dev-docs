# ColorHelpers.Transparent() Method

## Definition

Namespace: [Microsoft.CommandPalette.Extensions.Toolkit](microsoft-commandpalette-extensions-toolkit.md)

The **Transparent** method creates a color that represents transparency. This is useful when you want to specify a color that should not be visible, such as when you want to hide an element or make it invisible.

## Returns

An [OptionalColor](../microsoft-commandpalette-extensions/optionalcolor.md) object that represents transparency. This object can be used in various UI elements, such as command items and icons, to indicate that the element should not be visible.

# CommandProvider.FallbackCommands() Method

## Definition

Namespace: [Microsoft.CommandPalette.Extensions.Toolkit](microsoft-commandpalette-extensions-toolkit.md)

The **FallbackCommands** method retrieves a collection of fallback command items.

## Returns

An [IFallbackCommandItem[]](../microsoft-commandpalette-extensions/ifallbackcommanditem.md) array representing the fallback command items. This array contains the commands that are used as fallback options when no other commands are available or applicable in the command palette context.

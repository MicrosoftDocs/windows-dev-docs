# CopyTextCommand.Invoke() Method

## Definition

Namespace: [Microsoft.CommandPalette.Extensions.Toolkit](microsoft-commandpalette-extensions-toolkit.md)

Sets the clipboard text to the value of [Text](copytextcommand.md#properties) and returns the [Result](copytextcommand.md#properties).

## Returns

An [ICommandResult](../microsoft-commandpalette-extensions/icommandresult.md) object that represents the result of the command execution.

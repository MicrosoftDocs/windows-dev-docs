# ColorHelpers Class

## Definition

Namespace: [Microsoft.CommandPalette.Extensions.Toolkit](microsoft-commandpalette-extensions-toolkit.md)

The **ColorHelpers** class provides static methods for creating colors in the Command Palette SDK. It is used to create colors that can be used in various UI elements, such as command items and icons.

## Methods

| Method | Description |
| :--- | :--- |
| [FromArgb(Byte, Byte, Byte, Byte)](colorhelpers_fromargb.md) | Creates a color from the specified ARGB values. |
| [FromRgb(Byte, Byte, Byte)](colorhelpers_fromrgb.md) | Creates a color from the specified RGB values. |
| [NoColor()](colorhelpers_nocolor.md) | Creates a color that represents no color. |
| [Transparent()](colorhelpers_transparent.md) | Creates a color that represents transparency. |

# IDetailsCommands Interface

## Definition

Namespace: [Microsoft.CommandPalette.Extensions](microsoft-commandpalette-extensions.md)

The **IDetailsCommands** interface defines the commands to display on the details pane in the Command Palette.

## Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| Commands | [ICommand](icommand.md)[] | The commands to display on the details pane. |

# IFallbackCommandItem2 Interface

## Definition

Namespace: [Microsoft.CommandPalette.Extensions](microsoft-commandpalette-extensions.md)

The **IFallbackCommandItem2** interface extends [IFallbackCommandItem](ifallbackcommanditem.md) to add a unique identifier for the fallback command item. This interface requires [IFallbackCommandItem](ifallbackcommanditem.md).

## Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| Id | **String** | The unique identifier for the fallback command item. |

# IGalleryGridLayout Interface

## Definition

Namespace: [Microsoft.CommandPalette.Extensions](microsoft-commandpalette-extensions.md)

Requires: [IGridProperties](igridproperties.md)

The **IGalleryGridLayout** interface defines the layout properties for gallery grid items in the Command Palette.

## Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| ShowTitle | **Boolean** | Whether to show the title on gallery grid items. |
| ShowSubtitle | **Boolean** | Whether to show the subtitle on gallery grid items. |

# StatusContext Enum

## Definition

Namespace: [Microsoft.CommandPalette.Extensions](microsoft-commandpalette-extensions.md)

Specifies the context in which a status message is displayed in the Command Palette.

## Fields

| Field | Value | Description |
| :--- | :--- | :--- |
| Page | `0` | The status message applies to the current page. |
| Extension | `1` | The status message applies to the extension. |

# IMediumGridLayout Interface

## Definition

Namespace: [Microsoft.CommandPalette.Extensions](microsoft-commandpalette-extensions.md)

Requires: [IGridProperties](igridproperties.md)

The **IMediumGridLayout** interface defines the layout properties for medium grid items in the Command Palette.

## Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| ShowTitle | **Boolean** | Whether to show the title on medium grid items. |

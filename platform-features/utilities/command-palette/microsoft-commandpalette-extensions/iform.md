# IForm Interface

## Definition

Namespace: [Microsoft.CommandPalette.Extensions](microsoft-commandpalette-extensions.md)

The **IForm** interface represents a form in the Command Palette. It is used to define the properties and methods for creating and managing forms, such as displaying fields, handling user input, and validating data.

## Methods

| Method | Description |
| :--- | :--- |
| [DataJson()](iform_datajson.md) | The JSON representation of the data associated with the form. |
| [StateJson()](iform_statejson.md) | The JSON representation of the state associated with the form. |
| [SubmitForm(String)](iform_submitform.md) | Submits the form data. |
| [TemplateJson()](iform_templatejson.md) | The JSON representation of the template associated with the form. |

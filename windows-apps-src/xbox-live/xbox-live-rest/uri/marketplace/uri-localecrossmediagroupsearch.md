# /media/{marketplaceId}/crossMediaGroupSearch
Accesses items from several different media groups. 
The domain for these URIs is `eds.xboxlive.com`.
 
  * [URI parameters](#ID4EV)
 
<a id="ID4EV"></a>

 
## URI parameters
 
| Parameter| Type| Description| 
| --- | --- | --- | 
| marketplaceId| string| Required. String value obtained from the <b>Windows.Xbox.ApplicationModel.Store.Configuration.MarketplaceId</b>.| 
  
<a id="ID4EUB"></a>

 
## Valid methods

[GET (/media/{marketplaceId}/crossMediaGroupSearch)](uri-localecrossmediagroupsearchget.md)

&nbsp;&nbsp;Gets items from several different media groups.

# GetClipResponse (JSON)
Wraps the game clip. 
<a id="ID4EN"></a>

 
## GetClipResponse
 
The GetClipResponse object has the following specification.
 
| Member| Type| Description| 
| --- | --- | --- | 
| gameClip| [GameClip](json-gameclip.md)| A single game clip that satisfied the query.|
